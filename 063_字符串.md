### 字符串

在python中，字符串是序列，因此许多适用于序列的工具也适用于字符串

#### 访问字符串中的单个字符

##### 使用`for`循环

- 格式：`for character in string:`

- 适用场景：需要遍历整个字符串时（如统计特定字符的出现次数）。

##### 使用索引

- 个字符都有一个索引，用于指定其在字符串中的位置，索引从 0 开始。
- 格式：`character = my_string[i]`<img src="C:\Users\HP\AppData\Roaming\Typora\typora-user-images\image-20251026145222839.png" alt="image-20251026145222839" style="zoom:50%;" />

##### 示例

```python
# 使用for循环访问字符
my_string = "Hello"
for char in my_string:
    print(char)
# 输出：H e l l o

# 使用索引访问字符
print(my_string[0])  # 输出：H
print(my_string[3])  # 输出：l
```

##### `IndexError`异常

当尝试使用超出字符串范围的索引时，会引发`IndexError`异常。这种情况很可能发生在循环迭代超出字符串末尾时。

##### `len(string)`函数

`len(string)`函数可用于获取字符串的长度，有助于防止循环迭代超出字符串末尾。

#### 字符串拼接

拼接是指将一个字符串追加到另一个字符串的末尾。

##### 字符串拼接的操作方法

- **使用`+`运算符**：可生成由其操作数组合而成的字符串。
- **使用`+=`增强赋值运算符**：也可用于拼接字符串，需注意`+=`运算符左侧的操作数必须是已存在的变量，否则会引发异常。

##### 示例

```python
# 使用+运算符拼接
str1 = "Hello"
str2 = "World"
result = str1 + " " + str2
print(result)  # 输出：Hello World

# 使用+=运算符拼接
str3 = "Python"
str3 += " is fun"
print(str3)  # 输出：Python is fun
```

#### 字符串的不可变性

字符串是不可变的，一旦创建就无法被修改。

##### 字符串拼接的本质

拼接操作并不会实际修改已存在的字符串，而是创建一个新字符串，并将该新字符串赋值给之前使用的变量。

不能使用`string[index] = new_character`形式的表达式来修改字符串，此类语句会引发异常。

#### 字符串切片

切片是从序列中提取的一段项，对于字符串而言称为子字符串。

##### 切片的格式与功能

- 格式：`string[start : end]`
- 功能：返回一个包含从`start`索引开始（包含`start`）到`end`索引结束（不包含`end`）的字符副本的字符串。
- 缺省值：
  - 若未指定`start`，则起始索引为 0。
  - 若未指定`end`，则结束索引为`len(string)`。

##### 切片的扩展功能

切片表达式可包含步长值，也可使用相对于字符串末尾的负索引。

##### 示例

```python
my_string = "Hello World"
# 基本切片
print(my_string[0:5])  # 输出：Hello
print(my_string[6:])   # 输出：World（未指定end，默认到字符串末尾）
print(my_string[:5])   # 输出：Hello（未指定start，默认从0开始）

# 包含步长和负索引的切片
print(my_string[::2])  # 输出：HloWrd（步长为2）
print(my_string[-5:])  # 输出：World（负索引，从末尾第5个字符开始）
```

#### 判断字符串是否包含在另一个字符串内

##### `in`运算符

可使用`in`运算符判断一个字符串是否包含在另一个字符串中。

- 通用格式：`string1 in string2`
- `string1`和`string2`可以是字符串字面量或引用字符串的变量。

##### `not in`运算符的作用

类似地，可使用`not in`运算符判断一个字符串是否不包含在另一个字符串中。

##### 示例代码

```python
str1 = "Hello World"
str2 = "Hello"
str3 = "Python"

# 使用in运算符
print(str2 in str1)  # 输出：True
print(str3 in str1)  # 输出：False

# 使用not in运算符
print(str3 not in str1)  # 输出：True
```

#### 重复运算符的作用

重复运算符用于创建一个字符串的多个副本并将它们连接在一起。

##### 重复运算符的形式

- 符号`*`在应用于字符串和整数时是重复运算符。
- 字符串是左操作数，整数是右操作数。
- 通用格式：`string_to_copy * n`

##### 操作结果

变量将引用一个新字符串，该字符串包含原始字符串的多个副本。

##### 示例

```python
my_string = "Hello "
# 字符串重复操作
result = my_string * 3
print(result)  # 输出：Hello Hello Hello 
```

#### 拆分字符串

##### `split`方法的作用

`split`方法返回一个包含字符串中单词的列表。

#####  `split`方法的分隔符规则

- 默认使用空格作为分隔符。
- 可通过向`split`方法传递参数来指定不同的分隔符。

##### 示例

```python
# 默认分隔符（空格）
my_string = "Hello World Python"
result = my_string.split()
print(result)  # 输出：['Hello', 'World', 'Python']

# 指定分隔符（逗号）
my_string2 = "apple,banana,grape"
result2 = my_string2.split(',')
print(result2)  # 输出：['apple', 'banana', 'grape']
```
