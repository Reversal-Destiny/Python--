### 长语句换行

- 使用反斜杠 `\`
- 括号内换行可省略反斜杠

```python
var1, var2, var3, var4, var5 = 1, 2, 3, 4, 5
value1, value2, value3, value4, value5, value6 = 1, 2, 3, 4, 5, 6

# 使用反斜杠 \
result = var1 * 2 + var2 * 3 + \
         var3 * 4 + var4 * 5

# 在括号内换行（不用 \）
total = (value1 + value2 +
         value3 + value4 +
         value5 + value6)
```
