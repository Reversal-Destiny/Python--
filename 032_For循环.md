### For循环

for循环可以遍历任何序列的项目，如一个列表或者一个字符串。

#### 基本形式

```python
for iterating_var in sequence:
   statements(s)
```

#### 流程

![img](https://www.runoob.com/wp-content/uploads/2013/11/A71EC47E-BC53-4923-8F88-B027937EE2FF.jpg)

#### 示例

```python
for letter in 'Python':     # 第一个实例
   print("当前字母: %s" % letter)
 
fruits = ['banana', 'apple',  'mango']
for fruit in fruits:        # 第二个实例
   print ('当前水果: %s'% fruit)
 
print ("Good bye!")
"""
执行结果：
当前字母: P

当前字母: h
当前字母: o
当前字母: n
当前水果: banana
当前水果: apple
当前水果: mango
Good bye!
"""
```

#### Range函数
`range(a, b)`函数返回一个可迭代对象（iterable），使用这个函数可以简化编写`for`循环的过程

##### 基本语法

```python
range(start, stop[, step])
```

##### 参数说明：

- start: 计数从 start 开始。默认是从 0 开始。例如range（5）等价于range（0， 5）;

- stop: 计数到 stop 结束，**但不包括 stop**。例如：range（0， 5） 是[0, 1, 2, 3, 4]没有5

- step：步长，默认为1。例如：range（0， 5） 等价于 range(0, 5, 1)

**请注意，range(a, b)的范围为[a, b)，b不会被包含在范围内**

##### Range函数的形式

- 单值

  ```python
  for i in range(5):   # 0,1,2,3,4
      print(i)
  ```

- 双值

  ```python
  for i in range(2, 6):   # 第一个为起始值，第二个为结束值
      print(i)	# 2,3,4,5
  ```
  
- 三值

  ```python
  for i in range(1, 10, 2):   # 起始值，结束值，步长
      print(i)	# 1,3,5,7,9
  ```
  
  步长可以为**负数**，但要求起始值大于结束值
  
  ```python
  for i in range(10, 1, -1):	# 10,9,8,7,6,5,4,3,2
      print(i)

  for i in range(1, 10, -1):

  """
  ```
