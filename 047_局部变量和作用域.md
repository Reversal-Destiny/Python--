### 局部变量和作用域

| 要点            | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| 局部变量        | 是在函数内部被赋值的变量，属于其被创建的函数；只有该函数内部的语句可以访问它，若其他函数尝试访问会引发错误 |
| 作用域（Scope） | 是程序中变量可被访问的部分；对于局部变量，其作用域是被创建的函数内部 |

##### 局部变量特点

| 要点                 | 说明                                                         |
| -------------------- | ------------------------------------------------------------ |
| 局部变量的访问限制   | 函数内部，在局部变量创建之前的语句无法访问该变量             |
| 局部变量的命名独立性 | 不同函数可以有同名的局部变量，因为每个函数无法看到其他函数的局部变量，所以不会产生混淆 |
