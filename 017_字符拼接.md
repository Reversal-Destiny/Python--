### 字符拼接

可以使用`+`将两个字符串进行拼接：

```python
message = 'Hello ' + 'world'
print(message)  # 输出Hello world
```

#### 隐式拼接

- 相邻字符串字面量会自动拼接：

```python
my_str = 'one' 'two' 'three'
print(my_str)   # 输出onetwothree
```

#### Print的更多用法

- 默认结尾换行，可用 `end='分隔符'` 修改

- 默认参数间空格，可用 `sep='分隔符'` 修改

- 转义序列：`\n` 换行，`\t` 制表符

```python
# 1. 默认换行
print("Hello")
print("World")
# 输出：
# Hello
# World

# 2. 使用 end 参数（把换行改成空格）
print("Hello", end=" ")
print("World")
# 输出：
# Hello World

# 3. 使用 sep 参数（把默认的空格换成其他符号）
print("2025", "09", "20", sep="-")
# 输出：
# 2025-09-20

# 4. 转义序列 \n（换行）
print("Line1\nLine2\nLine3")
# 输出：
# Line1
# Line2
# Line3

# 5. 转义序列 \t（制表符）
print("A\tB\tC")
# 输出：
# A    B    C   （用制表符对齐）
```
