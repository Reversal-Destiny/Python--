### 字符串与字符串字面量

- **字符串 (string)**：由字符组成的数据

- **字符串字面量 (string literal)**：代码中出现的字符串

- 必须用 `'` 或 `"` 包裹；也可以用 `'''` 或 `"""` 包裹（支持多行）

#### 单引号和双引号

其中`'`和`"`功能相同,在大多数情况下是可以互换使用的，它们的核心功能都是定义字符串。两者的主要区别体现在如下场景：

**包含对方符号时无需转义**
如果字符串中包含单引号，使用双引号定义字符串更方便，无需转义：

```python
print("It's a cat")  # 直接包含单引号，无需转义
```
```python
print('He said "Hello"')  # 直接包含双引号，无需转义
```
