### 使用Python

在使用 Python 前，需要安装并配置 Python 环境。其中包括 **Python 解释器 (Python Interpreter)**

#### Python 解释器的两种模式

##### Interactive Mode （交互模式）
- 进入交互模式后，会看到提示符：>>>

- 提示符表示解释器正在等待输入 Python 语句。  

- 每执行完一条语句，提示符会重新出现。  

- 如果语句输入错误，会显示错误信息。  

- **适合用途**：学习和测试新的 Python 功能或语法。

##### Script Mode （脚本模式）

- 在交互模式下输入的语句不会保存为程序。  
- 如果要创建完整的程序，需要使用脚本模式：  
  - 将多条 Python 语句写入一个文件。  
  - 文件扩展名应为 `.py`。  
  - 运行方式：在操作系统的命令行输入或点击IDE的run按钮运行
