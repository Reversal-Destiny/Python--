### 循环

#### 功能

循环语句允许我们执行一个语句或语句组多次

<img src="https://www.runoob.com/wp-content/uploads/2015/12/loop.png" alt="img" style="zoom:70%;" />

Python 提供了 for 循环和 while 循环（在 Python 中没有 do..while 循环）:

| 循环类型                                                     | 描述                                                   |
| :----------------------------------------------------------- | :----------------------------------------------------- |
| [while 循环](https://www.runoob.com/python/python-while-loop.html)(Condition-controlled) | 在给定的判断条件为 true 时执行循环体，否则退出循环体。 |
| [for 循环](https://www.runoob.com/python/python-for-loop.html)(Count-controlled) | 重复执行语句                                           |
| [嵌套循环](https://www.runoob.com/python/python-nested-loops.html) | 你可以在while循环体中嵌套for循环                       |
