### 增强赋值运算符（Augmented Assignment Operators）

是一类用于简化赋值语句的运算符，在许多情况下，赋值语句左边的变量会同时出现在右边，例如 `x = x + 5`，这种写法既冗长又重复。增强赋值运算符提供了一种更简洁的方式，将运算和赋值合并在一起，如 `x += 5` 就等价于 `x = x + 5`。它们也被称为简写运算符，常见的形式包括 `+=`、`-=`、`*=`、`/=`、`//=`、`%=` 和 `**=`，分别对应加法、减法、乘法、除法、整除、取余和幂运算，使代码更简洁、可读性更高。

| 运算符          | 示例        | **等价语句**  |
| --------------- | ----------- | ------------- |
| +=              | x += 5      | x = x + 5     |
| negative equals | y minus = 2 | y = y minus 2 |
| *=              | z *= 10     | z = z * 10    |
| /=              | a /= b      | a = a / b     |
| %=              | c %= 3      | c = c % 3     |
