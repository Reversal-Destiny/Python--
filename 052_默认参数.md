### 默认参数

调用函数时，默认参数的值如果没有传入，则被认为是默认值。

```python
def printinfo( name, age = 35 ):	# 可以通过key = value定义默认参数
   "打印任何传入的字符串"
   print "Name: ", name
   print "Age ", age
   return
```

在函数的参数列表中，**无默认值的参数必须先出现**，之后再跟有默认值的参数。函数`def show_tax(price=10, tax_rate): ...`是无效的，因为有默认值的`price`参数出现在了无默认值的`tax_rate`参数之前，违反了上述顺序规则。



可以为函数的**所有参数**都提供默认值。例如函数`def show_tax(price=10, tax_rate=0.07): ...`，`price`和`tax_rate`都有默认值。此时我们只需要使用`show_tax()`即可调用函数(效果和`show_tax(10, 0.07)`相同)
