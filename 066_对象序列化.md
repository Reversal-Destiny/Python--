### 对象序列化

序列化是将对象转换为可轻松存储到文件的字节流。在 Python 中，通过`pickle`模块实现对象序列化的过程。

#### 操作流程

- **导入模块**：导入`pickle`模块。
- **打开文件**：以二进制写入模式（`wb`）打开文件。
- **调用`dump`函数**：使用`pickle.dump(object, file)`将对象序列化并写入文件（`object`为待序列化对象，`file`为打开的文件对象）。
- **关闭文件**：完成写入后关闭文件。
- **多对象序列化**：可在关闭文件前，多次调用`pickle.dump`将多个对象序列化到同一文件中。

```python
import pickle

data = {'name': 'Alice', 'age': 30}
nums = [1, 2, 3, 4]

# 序列化多个对象到文件
with open('data.pkl', 'wb') as f:
    pickle.dump(data, f)
    pickle.dump(nums, f)
```

#### 反序列化

反序列化是将字节流重新转换为对象的过程。

#### 反序列化步骤

- **导入模块**：导入`pickle`模块。
- **打开文件**：以二进制读取模式（`rb`）打开文件。
- **调用`load`函数**：使用`pickle.load(file)`从文件中反序列化对象（`file`为打开的文件对象）。
- **关闭文件**：完成读取后关闭文件。
- **多对象反序列化**：可在关闭文件前，多次调用`pickle.load`从同一文件中反序列化多个对象。

```python
import pickle

# 反序列化多个对象
with open('data.pkl', 'rb') as f:
    restored_data = pickle.load(f)
    restored_nums = pickle.load(f)

print(restored_data)  # 输出：{'name': 'Alice', 'age': 30}
print(restored_nums)  # 输出：[1, 2, 3, 4]
```
