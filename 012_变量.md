### 变量

#### 变量赋值

**Python 中的变量赋值不需要类型声明。**

```java
int localVar = 10;	//Java 变量需要先声明类型
```

```python
localVar = 10	# Python 赋值不需要声明类型
```

每个变量在内存中创建，都包括变量的**标识，名称和数据**这些信息。**每个变量在使用前都必须赋值**，变量赋值以后该变量才会被创建。等号 `=` 用来给变量赋值。

```python
variable_name = "variable value"	# 左边为变量名称(变量名不能加引号)右边为变量值
```

##### 多重赋值

```python
x, y, z = 0, 1, 2
```

#### 变量命名规则

- 不能是 Python **关键字**
- 不能有空格
- 必须以字母或下划线开头
- 大小写敏感
- 名称要能反映变量用途

#### 使用 `print` 显示多个项

- `print` 可接收多个参数，用逗号分隔
- 输出时会自动加空格
```python
x, y, z = 1, 2, 3
print(x, y, z)
# 输出 1 2 3
```

#### 变量重新赋值

- 变量可以在程序中引用不同的值
- **垃圾回收 (garbage collection)**：删除不再使用的值
- 变量可以引用任意类型

#### 变量基本类型

- Numbers（数字）：`int`，`float`, `long`, `complex`(复数)
- String（字符串）: `str`
- List（列表）: `list`
- Tuple（元组）：`tuple`
- Set（集合）：`set`
- Dictionary（字典）：`dict`

#### Python数据类型转换

有时候，我们需要对数据内置的类型进行转换，数据类型的转换，你只需要将数据类型作为函数名即可。

```python
num = 1	# 整型
float_num = float(num)	# 整型 -> 浮点
print(float_num)	# 输出1.0
```
