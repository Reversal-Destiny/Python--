### 仅位置参数

- 参数传递限制

  仅位置参数只能接收位置实参，不允许以关键字参数的形式传递。

- 语法声明

  在函数参数列表中插入正斜杠（/），正斜杠之前的所有参数即为仅位置参数。

- 作用

  这种机制用于明确参数的传递方式，避免调用者因误传关键字参数而引发逻辑错误，同时也能在一些底层或库函数设计中，保证参数传递的规范性和一致性。

```python
def show_sum(a, b, c, d, /):
    pass
```

这些参数只能通过**位置实参**传递，不允许以关键字参数的形式传递（例如调用时不能写`show_sum(a=1, b=2, c=3, d=4)`或者`show_sum(10, 20, 30, d=40)`，只能按位置传递`show_sum(1, 2, 3, 4)`）。

正斜杠（`/`）可出现在参数列表的**任意位置**，但只有出现在它**之前**的参数才会成为仅位置参数。

```python
def show_sum(a, b, /, c, d):
    pass
```

在此示例中`a`和`b`是仅位置参数，只能通过位置实参传递；`c`和`d`既可以通过位置实参传递，也可以通过关键字参数传递。

在 Python 中调用函数时，**不能在关键字参数之后传递位置参数**。以函数`show_sum(a, b, /, c, d)`为例，若对参数`c`使用关键字参数传递（如`c=3`），则参数`d`也必须使用关键字参数传递（如`d=4`）；不能在`c`用关键字传递后，`d`再用位置参数传递。
