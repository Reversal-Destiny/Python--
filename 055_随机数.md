### 随机数

- 随机数的用途

  随机数在诸多编程任务中非常有用，例如模拟、游戏、数据采样等场景。

- random 模块

  该模块包含了处理随机数的库函数，是 Python 标准库的一部分。

- 点表示法（Dot notation）

  用于调用模块所属的函数，格式为

  ```python
  module_name.function_name()
  ```

  例如调用`random`模块的函数时可写

  ```python
  import random
  random.randint()	# randint(a, b)	返回介于a和b之间(包含a, b)的随机整数
  ```

#### 伪随机数和种子

- 伪随机数特性

  `random`模块生成的随机数实际上是伪随机数，由特定算法生成。

- 种子值的作用

  种子值用于初始化生成随机数的算法公式。要获得不同的随机数序列，需使用不同的种子值。

  - 默认情况下，种子值由系统时间决定。
  - 可通过`random.seed()`函数指定自定义的种子值。

这种机制的作用是在需要可复现的随机数场景（如测试、调试）中，通过固定种子值来保证随机数序列的一致性；同时也支持在普通场景下由系统时间自动生成种子，保证随机性。
