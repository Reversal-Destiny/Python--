### 语法

#### If 语句

```python
if 判断条件：
    执行语句……
else：
    执行语句……
```

其中"判断条件"成立时（非零），则执行后面的语句，而执行内容可以多行，以缩进来区分表示同一范围。else 为可选语句，当需要在条件不成立时执行内容则可以执行相关语句。

```python
a = 0
if a == 0:
    print("a is zero")
else:
    print("a is not zero")
# 输出a is zero
```

##### 单行if语句

```python
if score > 59: print('You passed!')	# 无需换行
```



#### Elif语句

当判断条件为多个值时，可以使用以下形式：

```python
if 判断条件1:
    执行语句1……
elif 判断条件2:
    执行语句2……
elif 判断条件3:
    执行语句3……
else:
    执行语句4……
```

```python
a = 1
if a == 0:
    print("a is zero")
elif a == 1:
    print("a is one")
else:
    print("a is not zero, nor one")
# 输出a is one
```

#### If-elif-else v.s. If else

```python
# 示例 1: 使用 if-elif-else
score = 85

if score >= 90:
    print("优秀")
elif score >= 75:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
# 输出: 良好


# 示例 2: 使用 if-else（只能两种情况）
age = 18

if age >= 18:
    print("成年人")
else:
    print("未成年人")
# 输出: 成年人

```
