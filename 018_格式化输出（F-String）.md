### 格式化输出（F-String）

格式化字符串通常是由`f`开头的字符串(例如`f'Hello world'`)。格式化的字符内可通过添加`{var_name}`的方式引用变量：

```python
name = 'Johnny'
print(f'Hello {name}.')	# 输出Hello Johnny.
print(f'The value is {10 + 2}.')	# 输出The value is 12.
```

#### 格式说明符

| 数字       | 格式       | 输出      | 描述                         |
| :--------- | :--------- | :-------- | :--------------------------- |
| 3.1415926  | {num:.2f}  | 3.14      | 保留小数点后两位             |
| 3.1415926  | {num:+.2f} | +3.14     | 带符号保留小数点后两位       |
| -1         | {num:-.2f} | -1.00     | 带符号保留小数点后两位       |
| 2.71828    | {num:.0f}  | 3         | 不带小数                     |
| 5          | {num:0>2d} | 05        | 数字补零 (填充左边, 宽度为2) |
| 5          | {num:x<4d} | 5xxx      | 数字补x (填充右边, 宽度为4)  |
| 10         | {num:x<4d} | 10xx      | 数字补x (填充右边, 宽度为4)  |
| 1000000    | {num:,}    | 1,000,000 | 以逗号分隔的数字格式         |
| 0.25       | {num:.2%}  | 25.00%    | 百分比格式                   |
| 1000000000 | {num:.2e}  | 1.00e+09  | 指数(科学计数法)记法         |
| 100000         | {num::,.2f} | 100,000.00        | 千位分隔符          |
| 13         | {num:>10d} | 13        | 右对齐 (默认, 宽度为10)      |
| 13         | {num:<10d} | 13        | 左对齐 (宽度为10)            |
| 13         | {num:^10d} | 13        | 中间对齐 (宽度为10)          |
