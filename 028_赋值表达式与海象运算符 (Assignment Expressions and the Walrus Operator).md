### 赋值表达式与海象运算符 (Assignment Expressions and the Walrus Operator)

- **海象运算符 `:=`** 是一种增强的赋值运算符  
- 可以用海象运算符创建 **赋值表达式**  

#### 赋值表达式的作用
1. 给变量赋值  
2. 返回被赋值的结果  

#### 示例
```python
# 普通赋值
x = 10
print(x)   # 输出 10

# 使用海象运算符
print(y := 20)   # 输出 20，同时将 20 赋值给 y
```

#### 海象运算符的优先级
- 海象运算符在 Python 所有运算符中 **优先级最低**
- 当在包含多个运算符的复杂表达式中使用时，海象运算符会 **最后执行**
- 在大多数情况下，需要在赋值表达式外加括号，以确保海象运算符将正确的值赋给变量

#### 示例
```python
# 没有括号，可能产生错误结果
# result = x + y := 5   # Syntax error

# 正确写法：使用括号
result = (z := 5) + 10
print(result)  # 输出 15
print(z)       # 输出 5
```

- 赋值表达式 **不是一个完整的语句**  
  - 例如，单独写：
    ```python
    num := 99   # ❌ 会报错
    ```

- 它必须作为 **更大语句的一部分** 出现  
  - 正确写法：
    ```python
    print(num := 99)   # 输出 99，同时将 99 赋值给 num
    ```
