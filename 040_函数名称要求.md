### 函数名称要求

| 命名规则要点   | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| 不能使用关键字 | 例如在 Python 中，`if`、`for`、`def`等关键字不能作为函数名   |
| 不能包含空格   | 函数名需是连续的字符组合，若有多个单词可采用下划线连接（如`calculate_area`）或驼峰命名法（如`calculateArea`） |
| 首字符要求     | 必须是字母（大小写均可）或下划线（如`_get_data`、`Hello`）   |
| 其他字符要求   | 只能是字母、数字或下划线（如`add_123`）                      |
| 大小写区分     | uppercase 和 lowercase 视为不同，如`MyFunction`和`myfunction`是两个不同的函数名 |

函数名应能描述其执行的任务，通常包含动词（如`calculate_area`、`print_result`），便于开发者理解函数功能
