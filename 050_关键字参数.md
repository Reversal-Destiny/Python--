### 关键字参数

关键字参数和函数调用关系紧密，函数调用使用关键字参数来确定传入的参数值。使用关键字参数允许函数调用时参数的顺序与声明时不一致，因为 Python 解释器能够用参数名匹配参数值。

```python
def print_info( name, age ):
   "打印任何传入的字符串"
   print "Name: ", name
   print "Age ", age
   return
 
#调用print_info函数
print_info( age=50, name="miki" )
```

在函数的参数列表开头，写一个星号（`*`），后跟一个逗号，即可实现要求所有参数必须以**关键字参数**的形式传递。这种机制的作用是强制调用者明确指定参数名传递参数，提升代码的可读性和可维护性，避免因参数位置混淆导致的错误。

```python
def show_sum(*, a, b, c, d):
    pass
```

调用该函数必须使用如下形式：

```python
show_sum(a=1, b=2, c=3, d=4)
```

星号（`*`）可出现在参数列表的**任意位置**，但只有出现在它**之后**的参数才会成为仅关键字参数。

```python
def show_sum(a, b, *, c, d):
    pass
```

在此示例中`a`和`b`可通过位置或关键字传递，而`c`和`d`必须以关键字参数形式传递（如`show_sum(1, 2, c=3, d=4)`）。
